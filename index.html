About

My journey into DevOps started with a deep interest in how systems work behind the scenes — how code moves from development to deployment, and how automation, infrastructure, and monitoring come together to deliver reliable software. That curiosity led me to cloud computing and eventually to the AWS re/Start program, where I gained real-world experience working with cloud services, containers, and CI/CD workflows.

I’ve since developed strong skills in building and maintaining cloud-native systems using tools like Docker, GitHub Actions, and AWS services such as EC2, S3, IAM, and CloudWatch. I’m particularly passionate about solving inefficiencies in workflows and using automation to improve team productivity.

I am an advocate for self-driven learning, and someone who enjoys helping others break into tech through mentoring and knowledge sharing. I’m always exploring new ways to grow, both personally and professionally.
